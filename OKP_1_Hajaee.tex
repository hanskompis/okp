\section{Kielten syntaksi}
\subsection{Rakenne}
Näin tulostetaan Hello World!
\begin{verbatim}
import std.stdio;

void main() {
 writeln("Hello World!");
}
\end{verbatim}
C-kielestä tuttu tapa on mahdollinen...

\begin{verbatim}
import std.stdio;

void main() {
  for(int i=0; i<10; i++) {
    writeln("Rivi: ", i);
  }
}
\end{verbatim}
Iteraatio voidaan tehdä myös seuraavasti:
\begin{verbatim}
import std.stdio;

void main() {
  foreach(i; 0 .. 10) {
    writeln("Rivi: ", i);
  }
}
\end{verbatim}
